■これはなに
某大人気サンドボックスゲームに使われているブロックや鉱石、ツルハシ各種です。撮影用MODとしてお使いください。  

![image](https://github.com/routehachi/r8_CM3D2_Mod/blob/master/%E9%89%B1%E5%A4%AB%E3%82%BB%E3%83%83%E3%83%88/sample.jpg "5000兆円欲しい！")  

クラフティング機能は搭載してません。ご了承ください。  

* MODの場所：「服装」＞「トップス」
* ツルハシのみsizeシェイプキーを搭載しているので、ShapeAnimatorを導入していればサイズの調節が可能です。
* 当MODは、モデルやテクスチャデータをMagicaVoxelを用いて一から製作しています。
