# これはなに
Hearts様のPCゲーム「ナツイロココロログ」風の目＋目ハイライトです。  
無限色のみ対応しています。  

![image](https://github.com/routehachi/r8_CM3D2_Mod/blob/master/%E3%83%AD%E3%83%AD%E3%83%AD%E3%82%B0%E9%A2%A8%E7%9B%AE%EF%BC%8B%E3%83%8F%E3%82%A4%E3%83%A9%E3%82%A4%E3%83%88/sample1.jpg "ことねたそ＾〜")  

ヘルミさん作のフェイス「カスタムフェイス5号(HLM-05)」と相性が良いかもしれません。  
サンプル画像の睫毛はフェイスのtexを改変したもので、本MODには含まれておりません。ご了承ください。
