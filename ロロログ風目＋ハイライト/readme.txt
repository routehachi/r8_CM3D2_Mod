■これはなに
　Hearts様のPCゲーム「ナツイロココロログ」風の目＋目ハイライトです。
　無限色のみ対応しています。

　ヘルミさん作のフェイス「カスタムフェイス5号(HLM-05)」と相性が良いかもしれません。
　サンプル画像の睫毛はフェイスのtexを改変したもので、本MODには含まれておりません。ご了承ください。

■インストール
　「ロロログ風目＋ハイライト」フォルダをSybaris\GameData内にコピーしてください。

　MOD制作に慣れておらず不具合が出ることもあるかもしれません。
　もし何かありましたらTwitter(@grecha_85)宛にメッセージ頂ければ幸いです。
